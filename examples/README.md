# Examples

This directory contains reproducible connection and integration examples.

Planned examples:

- PuTTY raw TCP connection;
- Waveshare PoE serial-server configuration;
- ioBroker Pylontech adapter connection;
- parsing recorded responses;
- MQTT publication;
- Modbus TCP mapping.

Examples must use placeholder IP addresses and must not contain credentials. Hardware examples must clearly distinguish RS232 from RS485, RS422, CAN and Ethernet even where the same connector type is used.
